Load required R Libraries for Natural Language Processing and set working directory and file path.


Load data This routine uses data from a corpus called HC Corpora (www.corpora.heliohost.org). The readme file at

http://www.corpora.heliohost.org/aboutcorpus.html provides details on the corpora available. The files have been language filtered but may still contain some foreign text. This routine applies data science to natural language processing.

The first calculation reads a large twitter data file, a large news file, and a large blog collection.
